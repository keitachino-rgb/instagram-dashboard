## 実装概要

Vercel にデプロイ可能な、Next.js + TypeScript + Tailwind CSS を使用したフル機能のInstagram運用分析ダッシュボードです。

### 実装済み機能一覧

#### ✅ 分析指標（13個）

**基本指標**
- [x] フォロワー数推移

- [x] リーチ・インプレッション推移
- [x] 投稿別パフォーマンス（TOP 20）
- [x] ハッシュタグ分析（15個）

**新規追加指標**
- [x] 時間帯別エンゲージメント分析（朝・昼・夜・深夜）
