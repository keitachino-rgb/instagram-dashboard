# プロジェクト概要

## Instagram運用分析ダッシュボード

Vercel にデプロイ可能な、Next.js + TypeScript + Tailwind CSS で構築されたフル機能のInstagram運用分析ダッシュボードです。

## プロジェクト情報

| 項目 | 内容 |
|------|------|
| **プロジェクト名** | instagram-dashboard-vercel |
| **説明** | Instagram 運用分析ダッシュボード |
| **バージョン** | 1.0.0 |
| **Node.js** | 18.17.0+ |
| **npm** | 9.0.0+ |
| **デプロイ先** | Vercel |

## 実装された機能

### 1. 分析指標（13項目）

#### 基本指標
- ✅ フォロワー数推移
- ✅ エンゲージメント分析
- ✅ リーチ・インプレッション
- ✅ 投稿別パフォーマンス
- ✅ ハッシュタグ分析

#### 追加指標（新規実装）
- ✅ 時間帯別エンゲージメント分析（朝・昼・夜・深夜）
- ✅ 曜日別パフォーマンス（月～日の比較）
- ✅ プロフィール訪問数推移
- ✅ 保存数分析
- ✅ フォロー/アンフォロー推移
- ✅ アクティブユーザー数
- ✅ リーチ率（フォロワーに対する実リーチ%）
- ✅ 予測トレンド（簡易的な近似線）

### 2. UI/UX機能

- ✅ **ダークモード対応**: トグル機能で簡単切り替え
- ✅ **レスポンシブデザイン**: モバイル・タブレット・デスクトップ対応
- ✅ **ツールチップ機能**: 各指標の詳細説明
- ✅ **スムーズアニメーション**: Tailwind アニメーション
- ✅ **期間フィルタ**: 7日・14日・30日・60日・90日
- ✅ **データ比較**: 前期間比での成長率表示
- ✅ **ホバーエフェクト**: インタラクティブなフィードバック

### 3. エクスポート機能

- ✅ **PNG 画像出力**: ダッシュボード全体をスクリーンショット
- ✅ **PDF レポート生成**: 高品質な PDF 形式出力
- ✅ **JSON データエクスポート**: 詳細データのダウンロード

### 4. グラフとデータ可視化

実装されたグラフタイプ：
- ✅ 折れ線グラフ（フォロワー推移）
- ✅ 面積グラフ（エンゲージメント）
- ✅ 棒グラフ（比較分析）
- ✅ 複合グラフ（複数データの同時表示）

### 5. データ管理

- ✅ 90日分の日次データ
- ✅ 20件以上の投稿データ
- ✅ 15件以上のハッシュタグデータ
- ✅ 時間帯別データ
- ✅ 曜日別データ

## 主要機能の詳細

### 1. ダッシュボード表示

**KPI カード（8個）**
- フォロワー数
- 総いいね数
- 総リーチ
- エンゲージメント率
- 総コメント数
- 総保存数
- 総インプレッション
- プロフィール訪問数

**グラフ（6個）**
- フォロワー数推移（面積グラフ）
- エンゲージメント分析（棒グラフ）
- リーチ・インプレッション（折れ線グラフ）
- フォロー/アンフォロー（棒グラフ）
- 時間帯別エンゲージメント（棒グラフ）
- 曜日別パフォーマンス（折れ線グラフ）

### 2. 分析テーブル

**投稿パフォーマンステーブル**
- TOP 20 投稿のランキング
- いいね・コメント・保存数
- リーチ・インプレッション
- エンゲージメント率
- 投稿タイプ（投稿・リール・ストーリー）
- ソート機能

### 3. ハッシュタグ分析

- 15 個のハッシュタグをランキング表示
- 使用回数・リーチ・エンゲージメント率
- ビジュアルプログレスバー表示
- 統計情報（合計使用数・総リーチ・平均エンゲージメント）

### 4. 期間フィルタ

選択可能な期間：
- 7 日
- 14 日
- 30 日
- 60 日
- 90 日

### 5. エクスポート機能

**3 つのエクスポート形式**：
1. PNG 画像（html2canvas 使用）
2. PDF レポート（jsPDF 使用）
3. JSON データ（詳細データ）

### よくある問題

|------|------|--------|
| npm install が遅い | 遠いレジストリ | `npm config set registry https://registry.npmmirror.com` |
