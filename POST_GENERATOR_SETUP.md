# Instagram 投稿管理・生成機能 セットアップガイド

## 概要

Instagram ダッシュボードに、完全無料で使用できる投稿管理・生成機能を追加しました。以下の機能が含まれます：

- **投稿テキスト自動生成** - テンプレートベースの自動生成機能
- **画像取得** - Unsplash API を使用した無料画像検索
- **投稿管理** - ローカルストレージを使用した下書き・予約管理
- **ハッシュタグエディター** - 提案機能付きのハッシュタグ管理
- **Instagram風プレビュー** - 実際の投稿イメージを確認
- **レスポンシブUI** - モバイル対応のモダンなUI

## 機能詳細

### 1. テキストテンプレート

6つのカテゴリから選択可能：
- **新作発表** - 新商品のローンチ投稿
- **キャンペーン** - セールやキャンペーン情報
- **ユーザー紹介** - お客様の声やUGC
- **シーズン企画** - 季節限定のコンテンツ
- **日常投稿** - 日々のおすすめ商品
- **感謝投稿** - マイルストーン達成時の感謝

各カテゴリに複数のテンプレートがあり、「ランダム選択」機能で自動的に選ばれます。

### 2. 画像検索（Unsplash API）

キーワード検索で Unsplash から高品質な画像を自動取得できます：

**利点：**
- 完全無料（月50,000回まで）
- APIキーなしでも使用可能（ダミー画像を表示）
- 複数枚の画像から選択可能
- グリッド表示でプレビュー

**セットアップ（オプション）：**

1. https://unsplash.com/oauth/applications にアクセス
2. 「New Application」をクリック
3. アプリケーション詳細を入力して作成
4. 「Access Keys」セクションからアクセスキーをコピー
5. `.env.local` に追加：

```bash
NEXT_PUBLIC_UNSPLASH_API_KEY=your_access_key_here
```

### 3. 投稿管理機能

**ローカルストレージに保存：**
- 下書き投稿
- 予約投稿
- 投稿済み

**操作：**
- 新規作成
- 下書き保存
- 編集・更新
- スケジュール設定
- 削除

### 4. ハッシュタグエディター

**機能：**
- 手動入力による追加
- 提案ハッシュタグからの選択
- 削除・編集機能
- Instagramの推奨（最大30個）表示

### 5. スケジュール機能

- 日付・時刻選択
- 今後の投稿を予約
- 即時投稿も可能

### 6. プレビュー表示

- Instagram風のプレビュー
- テキスト + 画像の確認
- ハッシュタグの表示
- 予測エンゲージメント表示

## 使用方法

### 基本的な投稿作成フロー

1. **ダッシュボードを開く**
   - メインダッシュボードの「投稿管理・生成」セクションに移動

2. **テキストを選択または作成**
   - テンプレートから選択するか、手動で入力
   - 「ランダム選択」で自動選択も可能

3. **画像を検索・選択**
   - キーワードを入力して画像検索
   - 複数の画像から1つを選択
   - 画像なしでも投稿可能

4. **ハッシュタグを追加**
   - 提案から選択するか、カスタマイズ
   - 最大30個が推奨

5. **スケジュールを設定（オプション）**
   - 即時投稿か予約投稿か選択
   - 予約の場合は日時を指定

6. **保存または投稿**
   - 「保存」で下書き保存
   - 「今すぐ投稿」で即投稿

### 保存済み投稿の管理

1. **「保存済み投稿」タブを選択**
2. 投稿一覧から以下のアクション可能：
   - **編集** - 投稿内容を修正
   - **削除** - 投稿を削除

## ローカルストレージについて

投稿データはブラウザのローカルストレージに保存されます：

- **保存キー**: `instagram-dashboard-posts`
- **データ形式**: JSON
- **永続性**: ブラウザキャッシュをクリアするまで保存
- **容量**: 通常 5MB（ブラウザにより異なる）

### ローカルストレージの確認

ブラウザの開発者ツール > Application > Local Storage で確認できます。

## API 仕様

### Unsplash API

**エンドポイント:**
```
GET https://api.unsplash.com/search/photos
```

**パラメータ:**
- `query` (string): 検索キーワード
- `page` (number): ページ番号
- `per_page` (number): 1ページの件数
- `client_id` (string): Unsplash APIキー

**レート制限:**
- APIキーあり: 月50,000リクエスト
- APIキーなし: 時間制限あり（ダミー画像使用）

## テンプレートのカスタマイズ

```typescript
  // ...

```

## トラブルシューティング

### 画像が表示されない

1. **Unsplash APIキーが設定されていない場合**
   - ダミー画像が表示されます（正常動作）
   - APIキーを設定して実画像を使用可能

2. **キーワード検索で結果がない場合**
   - 別のキーワードを試してください
   - 英語のキーワードをお勧めします

### ローカルストレージが満杯

- ブラウザのキャッシュをクリア
- 不要な投稿を削除
- 別のブラウザを使用

### スタイルが反映されていない

```bash
npm run build
npm run dev
```

でビルドを再実行してください。
